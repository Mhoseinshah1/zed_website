📖 راهنمای اتصال تلگرام

  1. از @BotFather یک بات جدید بسازید و توکن را دریافت کنید.
  2. یک سوپرگروه با قابلیت Topics ایجاد کنید و بات را به‌عنوان ادمین اضافه کنید.
  3. Chat ID گروه را از طریق @getmyid_bot یا API دریافت کنید.
  4. توکن و Chat ID را در فرم بالا وارد کرده و ذخیره کنید.
  5. با دکمه تست اتصال صحت تنظیمات را بررسی کنید.
  6. با دکمه ساخت تاپیک‌ها تاپیک‌های سازمان‌یافته را ایجاد کنید.
