📖 راهنمای اتصال تلگرام

  1. از @BotFather یک بات جدید بسازید و توکن را دریافت کنید.
  2. یک سوپرگروه با قابلیت Topics ایجاد کنید.
  3. بات را به‌عنوان ادمین با دسترسی ارسال پیام و مدیریت تاپیک اضافه کنید.
  4. Chat ID گروه را از @getmyid_bot یا Telegram API دریافت کنید.
  5. توکن و Chat ID را در فرم بالا وارد کرده و ذخیره کنید.
  6. با دکمه تست اتصال صحت تنظیمات را تأیید کنید.
  7. با دکمه ساخت تاپیک‌ها تاپیک‌های فارسی را در گروه ایجاد کنید.
